# ULTRON Skills & Integrations

Esta pasta contém todas as Skills, MCPs (Model Context Protocol) e integrações externas do ULTRON.

## Estrutura de Pastas

### 📁 MCPs/
Contém os servidores MCP (Model Context Protocol) que fornecem funcionalidades específicas ao ULTRON:
- Protocolos de comunicação com serviços externos
- Ferramentas especializadas para diferentes domínios
- Conectores para APIs e bases de dados

### 📁 Integrations/
Contém as integrações com serviços externos:
- APIs de terceiros (Google, Microsoft, etc.)
- Serviços de nuvem (AWS, Azure, GCP)
- Ferramentas de produtividade
- Plataformas de comunicação

### 📁 Templates/
Contém modelos e templates para:
- Criação de novas Skills
- Configuração de MCPs
- Estruturas de integração padrão

## Como Adicionar uma Nova Skill

1. Crie uma pasta na categoria apropriada (MCPs ou Integrations)
2. Inclua os ficheiros necessários (código, configuração, documentação)
3. Atualize este README com informações sobre a nova skill
4. Teste a integração antes de fazer commit

## Convenções de Nomenclatura

- Use nomes descritivos em inglês
- Utilize kebab-case para nomes de pastas (ex: `google-drive-integration`)
- Inclua sempre um README.md em cada skill
- Documente as dependências e configurações necessárias
